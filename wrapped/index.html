First, create a Last.fm API account: www.last.fm/api/account/create
Copy and paste the API key to the respective field.

Second, create an app in the Spotify developer dashboard: developer.spotify.com/dashboard. Naming and other information is not important, add something like '127.0.0.1' to Redirect URIs.
This time, you will need to copy both client ID and client secret from the settings tab.

‚ùîA quick note on why we need Spotify API here

This app counts Listened Minutes as the sum of "times scrobbled * duration" for each track. The tags that a music player provides to Last.fm have to align with how the track is stored in their database (that includes artist, album and track title tags), otherwise there won't be a duration record for that track. That's where Spotify comes in handy - the app attempts to retrieve the duration from Spotify for each track where it's missing.

Source code is available on Github: vulpeace/last.fm-wrapped
